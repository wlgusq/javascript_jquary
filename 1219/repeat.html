제어문_03반복문(while, do while, for)

반복문은 작업을 반복수행할 때 사용하며, 조건문(if)이 프로그램을 지적으로 만든다면 반복문은 프로그램을 강력하게 만드는 역할을 한다.

주의할 점 - 조건을 잘못 설정하며 무한루프에 빠지는 오류가 생기며, 브라우저에 화면 멈춤현상이 발생됨.

프로그램 강제종료 : ctrl + alt + del 키를 눌러서 작업관리자를 띄워 프로그램을 강제종료한다.

  • while문 - 반복조건
  • do while문 - 반복조건
  • for문 - 반복회수
    1. while문
    while(조건식){
      실행할 내용;
    }

    2. do while문
    do{
      실행할 내용
    }
    while(조건식)

    3. for문
    for(초기값;조건식;증감식){
      실행할 내용;
    }
