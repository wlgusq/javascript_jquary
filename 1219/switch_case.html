제어문_02switch_case(선택문)

선택문(switch_case)은 사용자가 만든 조건이 만족할 때까지 계속 실행하면서 조건에 맞으면 실행되고 프로그램이 그 자리에서 멈춤.

선택문은 조건식의 값이 정확히 일치해야할 때 사용하며 이것이 if문과의 차이점이다.

break문 - 조건식에서 원하는 값이 일치할 때 프로그램이 현 위치에서 멈추게 하기 위한 명령어

    switch(변수){
      case 1 : 실행할 내용;break;
      case 2 : 실행할 내용;break;
      case 3 : 실행할 내용;break;
      default : 실행할 내용;
    }

실습1. 사용자가 입력한 숫자를 '홀수', '짝수'인지 판단하기

2. 등수별 상품 출력 - 사용자가 입력한 번호를 통해 등수별 상품이 출력됨.

    1등 : 당첨 ! 해외상품권
    2등 : 냉장고
    3등 : 상품권
    4등 : 꽝!

3. 현재 날짜정보를 통해 요일표시하기

시간객체(Data)를 활용하여 요일번호를 구하고 switch case문을 통해 한글명 '요일'을 출력해본다.
