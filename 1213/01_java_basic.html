자바스크립트 소개와 기초문법

  • 자바스크립트는 html문서 안에서 동작되는 언어이다.
  • 자바스크립트는 모든 명령어가 객체로 이루어진 언어이다.(html-태그, 속성, css-선택자, 속성, js-객체, 속성, 메서드)
  • html 문서의 head, body영역에 작성한다.
  • 'script'로 시작하여 '/script'로 닫아준다.
  • 자바스크립트는 주석을 작성할 때 2가지 종류가 있다. 한줄주석 '//주석내용' 두 줄 이상 주석 '/* 주석내용 */'
  • html, css는 대소문자를 구별 안하지만, 자바스크립트는 대,소문자를 구별한다.
  • 자바스크립트에서 문자데이터를 입력하고자 할 때는 ''(홀따옴표, 싱글쿼테이션), ""(쌍따옴표, 더블쿼테이션) 기호를 사용한다.
  • 자바스크립트는 css언어와 같이 내부스크립트, 외부스크립트가 있다.
  • 자바스크립트의 문서확장자는 '파일명.js'로 끝난다.
